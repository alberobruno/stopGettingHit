 <h2>Getting Started</h2>

<b>Prerequisites</b>

- You will need a parsed SSBM slippi file (SLP).

<b>Installation</b>

1.  Clone the repo onto your local machine

```sh
 git clone https://github.com/alberobruno/stopGettingHit.git
```

2.  Install dependencies

```sh
yarn install
```

3.  Run the app

```sh
yarn run dev
```
